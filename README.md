# 岩手県青色申告会連合会 ホームページ Ver1.0

このデータは、現在のデザインを崩さずに固定した安定版です。

## GitHub Pagesへの反映手順

1. このZIPを解凍します。
2. 中身だけを、GitHub Desktopで管理している `iwate-aoiro` フォルダへ上書きします。
3. GitHub Desktopを開きます。
4. Summary に `Ver1.0 安定版に更新` と入力します。
5. `Commit to main` を押します。
6. `Push origin` を押します。
7. 1〜3分待ってから公開URLを確認します。

## 注意

- `Pull origin` は押さないでください。
- `.git` フォルダは絶対に削除しないでください。
- 公開URLが404の場合は、GitHub Actionsの「Pages build and deployment」がSuccessになるまで待ってください。
